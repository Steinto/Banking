PROJECT TITLE: Kawaii Banking

PURPOSE OF PROJECT: the front end a bank teller would use at Kawaii bank

VERSION or DATE: 31/03/2025

HOW TO START THIS PROJECT: with blueJ installed run the 'package' BlueJ Project File. Then right click 'Main' class
and select void main(String[] args) then select ok

AUTHORS: Toby Steiner

USER INSTRUCTIONS: input in the text box at the bottom of the screen, read all onscreen text before
inputting answer, Fullscreen for best viewing experience. some example customers in the database are 'Toby Steiner', 'Atticus Stinky' and 'James Bond'.

LIMITATIONS: one customer cant have multiple accounts, no support for hyphens in names, no support for '/' or letters in address number.
